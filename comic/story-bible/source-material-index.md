## Usage notes
- The design sheets are copied into each character's `refs/` folder as the primary visual lock.
- The two Word documents were interpreted into markdown files under `comic/story-bible/` and `comic/characters/`.
- When later reference packs or revised model sheets arrive, add them as new files in each character's `refs/` directory instead of replacing the canon notes.
- Chapter-specific environment drawings should be copied into that chapter's `scene-refs/` folder instead of the global `comic/scenes/` library.
